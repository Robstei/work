Swad Logifle:

Configuration1: Konfiguration bzgl. Reihenfolge von geteiltem, selektivem und kombiniertem Run
rundid1: Angabe ob es sich um geteilten, selektiven und kombinierten Run handelt
blockid1: Block innerhalb des entsprechenden Runs
form1: Angezeigte Form
character1: Angezeigter Character
seperateAttention1:	1: Ein Char ist das Target in diesem Block		
			2: Eine Form ist das Target in diesem Block
			3: Ein Char und eine Form sind Target in diesem Block

target1:	0: Kein Target in diesem Stimulus
		1: Die Farbe ist ein Target in diesem Stimulus
		2: Die Form ist ein Target in diesem Stimulus

time1: Anzeigezeit des Stimulus in Millisekunden nach Start des Paradigmas
code2: Button der Gedrückt wurde 1: Die Taste "L", die Taste für Character Targets
				 2: Die Taste "S", die Taste für Form Targets
time2: Absolute Reaktionszeit in Millisekunden nach Start des Paradigmas
time_diff: Reaktionszeit
rm_type: rm_hit: auf ein Target korrekt reagiert
	 rm_miss: auf ein Target nicht reagiert
	 rm_incorrect: auf ein Target mit der falschen Taste reagiert
	 rm_false_alarm: auf ein Nontarget reagiert
	 rm_other: auf ein Nontarget nicht reagiert
subject1: subjectid die am Anfang eingegeben wurde
